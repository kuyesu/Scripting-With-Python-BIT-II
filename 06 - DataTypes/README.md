### Python DataTypes

## types of DataTypes
 - Text dataType
    - Strings (str)

- Numeric dataTypes
    - integers
    - floats
    - complex numbers

- Sequence dataTypes
    - list 
    - tuple
    - range

- Mapping datatypes
    - dictionary (dict)

- Set datatypes
    - set
    - frozenset

- Boolean datatype
    - bool (by default return True or False)

- Binary datatypes
    - bytes
    - bytearray
    - memoryview
